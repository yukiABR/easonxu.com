# easonxu.com (GitHub Pages)

## Files
- `index.html`: homepage
- `CNAME`: custom domain (`easonxu.com`)

## Publish
1. Create a public GitHub repository named `easonxu.com` (or any name).
2. Upload all files in this folder to the repository root.
3. In repository settings: `Pages` -> `Build and deployment`:
   - Source: `Deploy from a branch`
   - Branch: `main` and `/ (root)`
4. Wait 1-3 minutes and open the Pages URL.
5. In `Pages`, make sure custom domain is `easonxu.com`.
